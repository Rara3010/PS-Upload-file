<h1> FILE HOST TEMPORARY </h1>
<h2>Fitur Utama</h2>

<h3>ğŸ“ Manajemen File</h3>
<p>
Sistem File Host Temporary menyediakan fitur manajemen file sederhana yang
memungkinkan pengguna mengunggah file tanpa proses registrasi. File yang
diunggah akan tersimpan secara sementara dan otomatis dihapus setelah
melewati batas waktu yang ditentukan.
</p>
<ul>
  <li>File otomatis kedaluwarsa setelah <strong>3 jam</strong></li>
  <li>Mendukung unggahan file berukuran besar hingga <strong>128 MB</strong></li>
</ul>

<h3>ğŸ•“ Riwayat Upload</h3>
<p>
Sistem menyimpan riwayat unggahan secara lokal di sisi klien menggunakan
penyimpanan browser. Fitur ini membantu pengguna untuk kembali mengakses
tautan file yang telah diunggah apabila lupa menyimpan URL hasil unggahan.
</p>
<ul>
  <li>Riwayat unggahan disimpan di browser pengguna</li>
  <li>Pengguna dapat melihat kembali file yang pernah diunggah</li>
</ul>
